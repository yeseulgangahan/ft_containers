# ft_containers
C++ 98 STL(Standard Template Library)의 vector, map, stack, set의 구조와 기능을 재구현한 라이브러리입니다.

## Features
- C++ 98에 구현되어 있는 함수만 구현 (deprecated된 함수 포함)
- 모든 멤버함수, 비멤버함수, 오버로드 구현
- STL container와 같거나 빠른 처리 속도

## Implements
- vector
- map
- stack
- set
- std::iterator_traits
- std::reverse_iterator
- std::enable_if
- std::is_integral
- std::equal
- std::pair
- std::make_pair
